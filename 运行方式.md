# Beeline

    Beeline 要与HiveServer2配合使用
    服务端启动hiveserver2
    客户的通过beeline两种方式连接到hive
    1、beeline -u jdbc:hive2://localhost:10000/default -n root
    2、beeline
    beeline> !connect jdbc:hive2://<host>:<port>/<db>;auth=noSasl root 123
    默认 用户名、密码不验证
